# Features
* 회원가입

* 로그인 / 로그아웃

* 회원 정보 수정 

* 대쉬보드 기능

* 관리자 기능
  *  신청받은 대여 목록 확인 및 승인 거부 
  * 대여 된 상품 확인 및 반납 기능 
  * 연체 된 상품 확인 및 반납
  * 유저정보 리스트 확인 및 검색 
  
* 유저 기능
  * 게임 리스트 확인
  * 게임 상세정보 확인
  * 대여 이력 조회 
  
  
# USED SKILL
  * SpringFrameWork
  * JAVA 
  * JDBC
  * JAVASCRIPT (JQuery, Ajax)
  * HTML
  * CSS3
  * BootStrap
  * ORACLE
